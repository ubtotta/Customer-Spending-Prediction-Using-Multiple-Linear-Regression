# Customer-Spending-Prediction-Using-Multiple-Linear-Regression
Machine Learning project that predicts annual customer spending for an e-commerce business using Multiple Linear Regression. The project includes feature selection, correlation analysis, train-test splitting, model training, and performance evaluation using R² Score, MAE, and RMSE.

# 🛒 Ecommerce Customer Spending Prediction
# Project Overview

- This project predicts yearly customer spending based on customer engagement metrics such as session length, app usage, and membership duration. Multiple Linear Regression is used to model customer spending behavior.

# Technologies Used

- Python
- Pandas
- NumPy
- Scikit-Learn

# Features Used

- Avg. Session Length
- Time on App
- Length of Membership
- Target Variable
- Yearly Amount Spent

# Workflow

- Data Loading
- Data Cleaning
- Correlation Analysis
- Feature Selection
- Train-Test Split
- Multiple Linear Regression
- Model Evaluation
- R² Score
- MAE
-RMSE

# Results

The model achieved strong predictive performance and successfully explained a significant portion of customer spending behavior. Evaluation metrics demonstrated good generalization on unseen data.
